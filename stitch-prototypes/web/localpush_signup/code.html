Did you know?

Your Mac is a treasure trove of structured data hidden behind system barriers. LocalPush makes it accessible via standard APIs.

Apple Podcasts

Apple Podcasts stores your complete listening history — every episode, play times, how far you got — and full transcripts. All in a SQLite database on your Mac. LocalPush it.

Claude Code

Claude Code stores every session name, project, git branch, message counts, and daily token spend by model. Your AI work history lives in ~/.claude/. LocalPush it.

Notes & Knowledge

Every note you've written — created, last touched, which folder — tracked in a database on your Mac. LocalPush it.

Photos Library

Your Photos library tracks metadata on every image — when, where, camera, what's in it. Thousands of data points. LocalPush it.
